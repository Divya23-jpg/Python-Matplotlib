Subplots:
    In Matplotlib, plt.subplots() is a function that creates a figure and a grid of axes (subplots) in one call.
    It’s the most convenient way to generate multiple plots within the same figure.


Use  Of Subplots:
    1.To Comapre multiple charts side by side.
    2.Reduce space by Comapre multiple charts in one window no need to make another window for different charts
    3.Quickly Analyse data like Weekly sales , Ml model etc


Function of subplots:

1.plt.subplots():
    1.Object Oriented
    2.Preferred Method

2.plt.tight_layout():
    1.Automatic Adjustment
    2.Overlap Fix

3.ax.text(), ax.annotate():
    1.Text Addition
    2.Annotation Insertion

4.plt.subplot():
    1.Grid Layout
    2.Quick Creation

5.Sharex=True,Sharey=True:
    1.Shared X Axis
    2.Shared Y Axis
